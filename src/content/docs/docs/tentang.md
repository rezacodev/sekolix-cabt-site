Sekolix CABT adalah sistem **Computer Assisted Based Test (CAT)** berbasis web untuk keperluan ujian sekolah.

## Fitur Utama

- **6 tipe soal** — PG, PG Berbobot, PG Majemuk, Menjodohkan, Isian, Uraian
- **Timer server-authoritative** — tidak bisa dimanipulasi dari client
- **Resume ujian otomatis** — peserta bisa pindah komputer tanpa kehilangan jawaban
- **Anti-kecurangan multi-lapis** — server-side, client-side, dan jaringan
- **Laporan lengkap** — ekspor Excel, cetak daftar hadir, berita acara, kartu peserta
- **Livescore real-time** — leaderboard peserta polling 15 detik

## Level Pengguna

| Level | Peran | Akses |
|---|---|---|
| 1 | **Peserta** | Antarmuka ujian di `/peserta` |
| 2 | **Guru** | Panel admin terbatas — kelola soal, pantau sesi rombel |
| 3 | **Admin** | Kelola semua user, rombel, sesi, laporan |
| 4 | **Super Admin** | Semua akses + General Setting konfigurasi sistem |
