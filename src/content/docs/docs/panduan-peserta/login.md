## Login

1. Buka halaman aplikasi di browser
2. Masukkan **username atau email** dan **password**
3. Klik **Masuk** — sistem akan redirect otomatis ke dashboard peserta

:::note
Jika login gagal 5 kali berturut-turut, akun akan terkunci sementara selama 15 menit.
:::

## Dashboard Peserta

Setelah login, Anda akan melihat daftar sesi ujian yang tersedia. Setiap sesi menampilkan:
- Nama paket ujian
- Durasi ujian
- Status sesi (Aktif / Belum dibuka / Selesai)

## Memasuki Sesi Ujian

1. Klik sesi ujian yang ingin dikerjakan
2. Halaman konfirmasi akan muncul — periksa detail ujian (nama, durasi, jumlah soal)
3. Masukkan **token akses** yang diberikan oleh pengawas/guru
4. Klik **Mulai Ujian**

:::caution
Pastikan koneksi internet stabil sebelum memulai. Jawaban tersimpan otomatis, tetapi timer terus berjalan.
:::
